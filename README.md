# Yingo - IELTS Vocabulary MVP

Mobile-first web app for IELTS core vocabulary practice with pronunciation and spaced repetition.

## Stack
- Next.js 14, React 18, TypeScript
- Tailwind CSS + Lucide icons
- Web Speech API for pronunciation
- SM-2 algorithm for spaced repetition

## Features
- IELTS core word lists with example sentences
- Pronunciation guide and phoneme visualization
- Reading and writing practice (placeholder)
- Search and filter words by category
- User profile with progress tracking

## Getting Started
```bash
npm install
npm run dev
```
Visit `http://localhost:3000`.

## Project Structure
- `src/app/` - Next.js app router pages
- `src/components/` - UI components
- `src/data/` - Vocabulary and example datasets
- `src/lib/` - Utilities including SM-2 and TTS helpers
- `src/types/` - TypeScript interfaces
