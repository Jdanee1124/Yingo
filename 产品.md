# Yingo 产品开发记录

## 1. 产品概述
**Yingo** 是一款面向雅思（IELTS）备考者的移动端优先（Mobile-first）词汇学习应用。
核心理念：**移动优先，简洁实用，专注核心词汇**。

## 2. 开发目标
- 从0到1开发 MVP（最小可行性产品）
- 优先适配手机端，PC端兼容
- 积累雅思核心词汇，提供发音、例句、记忆技巧

## 3. 核心功能 (MVP)
### 3.1 词汇学习
- 雅思核心词库：当前包含 1485 个核心词汇
- 单词详情：音标、中文释义、英文释义、例句、词根联想
- 词汇分类：教育、科技、环境、健康等 12 大类
- 搜索功能：支持单词快速搜索

### 3.2 发音与语音
- 单词发音：基于 Web Speech API
- 音标学习：IPA 音标展示、分类视图、交互详情
- 发音技巧：连读、弱读、重音等技巧提示

### 3.3 记忆辅助
- SM-2 算法：基于间隔重复的记忆曲线管理
- 学习进度：每日统计、连续学习天数

## 4. 技术架构
- **前端框架**: Next.js 14 (App Router)
- **UI 样式**: Tailwind CSS + Lucide Icons
- **数据存储**: LocalStorage (MVP 阶段)
- **语音技术**: Web Speech API
- **包管理**: npm
