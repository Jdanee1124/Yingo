# Yingo 开发日志

## 1. 项目概述 (Project Overview)
**更新时间**: 2024-05-31

**Yingo** 是一款面向雅思（IELTS）备考者的移动端优先（Mobile-first）词汇学习应用。
- **核心理念**: 移动优先，简洁实用，专注核心词汇
- **仓库地址**: https://github.com/Jdanee1124/Yingo
- **分支**: main

## 2. 当前状态 (Current Status)
**版本**: MVP v1.0
**词汇量**: 1,892 个雅思核心词汇
**功能列表**:
- 词汇列表与详情 (音标、释义、例句)
- 搜索与分类筛选
- 发音播放 (Web Speech API)
- 音标学习 (网格视图与详情)
- 学习进度追踪 (SM-2 算法)

## 3. 技术架构 (Tech Stack)
- **框架**: Next.js 14 (App Router)
- **语言**: TypeScript
- **UI**: Tailwind CSS + Lucide Icons
- **数据**: LocalStorage (前端持久化)
- **语音**: Web Speech API
